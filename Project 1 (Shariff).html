The Components of the World Wide Web

We use the internet everyday without giving it much thought, especially on how it works. The internet has these essential components shown below in order for it to work:

Servers

Servers are computers that serve one purpose: to transmit, or serve files over the Web. Many of the websites today are being put out there on the Web via these servers.

Internet

The Internet is a network of servers inter-connecting each other to form the World Wide Web.

Browser

In order for us to view a page from a website, a software known as a browser is needed. The browser decodes the files of any website, or a webpage and these pages are created using the code known as HTML, or Hyper Text Mark Up Language.

HTTP

HTTP, or HyperText Transfer Protocol, is a computer communication protocol used for most communication on the Internet. It is a set of rules that conduct the client, i.e. web browser, and server interaction.

HTML Elements

For a html file to be displayed on the web browser, a computer language called HMTL, or Hyper Text Markup Language, is used. Inside of these files, we use codes known as the HTML elements, a set of tags which displays content on a web page. These elements consists of closing and an opening tags.
For example, to emboldened a text, we would use the <b>(opening) tag and </b>(closing)tag like this: <b>BOLD</b> Some tags do not require this tag: <../> -- a closing tag. These tags are known as void tags. These tags do not require closing tags since they can't contain content.

HTML Attributes

Some opening tags accepts additional values in them.These provides additonal information for the web browser. Two types of these attribute tags I learnt are the Hyperlink tag and the image tags.
A hyperlink tag looks like this: <a href="http://www.Iknowalready.com"> text here</a>, where as the tags used to diplay images are like this one <img src="image file name" alt="text">.
